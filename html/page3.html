Visual Programming برمجة مرئية

الدرس الثالث | lesson three

خطوات لإنشاء قاعدة بيانات ملقم SQL
إنشاء تطبيق Visual C#. NET Windows جديدة.
ضع زر على Form1. تغيير خاصية اسم الزر إلى بتنكريتيداتاباسي، ثم قم بتغيير الخاصية Text إنشاء قاعدة بيانات.
استخدام عبارة using على مساحات و نظام System.Data حيث لم تكن لتأهيل التعريفات بمساحات لاحقاً في التعليمات البرمجية الخاصة بك. أضف التعليمات البرمجية التالية إلى المقطع تعريفات عامة
التبديل إلى طريقة العرض، و إنشاء قاعدة بيانات لإضافة معالج حدث انقر فوق ثم انقر نقراً مزدوجاً فوق.
تغيير سلسلة الاتصال للإشارة إلى جهاز كمبيوتر يستخدم SQL Server ومن ثم تحقق من تعيين إلى رئيسي الوسيطة قاعدة البيانات أو فارغاً.
اضغط على F5 أو CTRL + F5 لتشغيل المشروع، ومن ثم انقر فوق إنشاء قاعدة بيانات.
استخدم "مستكشف الخادم" للتحقق من إنشاء قاعدة البيانات.
